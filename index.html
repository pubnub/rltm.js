Get Rltm.js

Why rltm.js?

Swap between Socket.io and PubNub with a single API.

Feature parity for different realtime services.

Works on web and NodeJS.

Config Examples

Socket.io

let client = rltm({
    service: 'socketio', 
    config: {
        endpoint: 'http://localhost:8000'
    }
});

PubNub

let client = rltm({
    service: 'pubnub', 
    config: {
        publishKey: 'YOUR_PUBNUB_PUBLISH_KEY',
        subscribeKey: 'YOUR_PUBNUB_SUBSCRIBE_KEY'
    }
});
